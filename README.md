# Swiggy SQL Data Analysis Project

## Project Overview
This project analyzes Swiggy food delivery data using SQL Server.

## Skills Used
- SQL
- Data Cleaning
- Data Validation
- Star Schema Design
- KPI Analysis
- Aggregations
- Joins
- CTEs
- Window Functions

## Key Insights
- Monthly order trends
- Revenue analysis
- Top restaurants
- Most ordered dishes
- Category performance
- Rating distribution

## Database Design
- Fact Table
- Dimension Tables
- Data Warehouse Model

## Tools Used
- SQL Server
- SSMS
